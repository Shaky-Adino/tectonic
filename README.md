# tectonic

A social media web application where people can connect with each other, share their thoughts, upload photos, create/delete posts, like/comment on others posts and have one on one chat with other users.

### Techstack

NextJS (ReactJS)                                                                                                                                                                   
MongoDB                                                                                                                                                                            
NodeJS                                                                                                                                                                             
Express                                                                                                                                                                           
SocketIO
